# Data Professionals Survey Dashboard

## Overview
This project presents an interactive Power BI dashboard built to analyze
survey responses from data professionals across different countries.

## Tools Used
- Power BI
- Microsoft Excel

## Key Insights
- Data Scientists earn the highest average salaries among data roles.
- Python is the most preferred programming language.
- Work-life balance satisfaction is higher than salary satisfaction.
